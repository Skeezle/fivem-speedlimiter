\# skeezle_speedlimiter by Skeezle scripts

Server-wide vehicle speed limiter for FiveM (Qbox / QBCore compatible) with configurable \*\*MPH or KPH\*\* limits, optional per-class and per-model overrides, and optional job exemptions.

---

\## Features


\- ✅ \*\*Global default speed limit\*\* (applies to all vehicles)

\- ✅ Configure in \*\*MPH\*\* or \*\*KPH\*\*

\- ✅ Optional \*\*per-vehicle-class\*\* limits

\- ✅ Optional \*\*per-vehicle-model\*\* (spawn name) overrides

\- ✅ Optional \*\*job exemptions\*\* (Qbox jobs, with QBCore fallback)

\- ✅ Optional exemption when \*\*siren is on\*\*

\- ✅ Optional \*\*toggle command + keybind\*\*

\- ✅ Uses \*\*player state bags\*\* to keep client behavior smooth


---


\## Requirements

\- Works on \*\*Qbox (qbx\_core)\*\* servers

\- Will also attempt a \*\*qb-core\*\* fallback for job detection

\- Optional:

&nbsp; - `ox\_lib` (only for nicer notifications; script still works without it)
